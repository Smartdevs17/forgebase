# ForgeBase

**ForgeBase** is an operational workbench for smart contracts on **Base** â€” enabling teams to safely simulate, compose, and execute complex contract interactions without writing scripts.

It provides a generic, Postman-like interface for interacting with any verified smart contract on the Base blockchain (Mainnet & Sepolia).

## Project Structure

The project is divided into two main components:

- **[Frontend](./frontend)**: A React/Vite application that provides the user interface.
- **[Backend](./backend)**: An Express/Node.js API that handles transaction decoding, event analysis, and contract metadata execution.

## Features

- **Contract Loader**: Instantly load any verified contract from Basescan.
- **Visual Interface**: Interact with read/write functions using autogenerated forms.
- **Workspace**: Save and organize your frequently used contract calls.
- **Base Integration**: Built specifically for the Base ecosystem (L2).
- **Simulation**: Preview transaction results before execution.

## Getting Started

1. **Clone the repository**:
   ```bash
   git clone <repository_url>
   cd forgebase
   ```

2. **Setup Backend**:
   ```bash
   cd backend
   npm install
   cp .env.example .env
   # Update .env with your RPC URLs (see backend/README.md)
   npm run dev
   ```

3. **Setup Frontend**:
   ```bash
   cd frontend
   npm install
   npm run dev
   ```

4. **Open in Browser**:
   Navigate to `http://localhost:5173` to start using ForgeBase.

## Technology Stack

- **Frontend**: React, TypeScript, TailwindCSS, Wagmi, Viem, shadcn/ui
- **Backend**: Node.js, Express, Ethers.js, MongoDB
